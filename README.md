# Regression-Analysis-Study
회귀분석 공부하자

## 공부 목적

그간 깔봤던 구조방정식이 모두 회귀분석에 기초한 것이었다. 통계 분석의 응용이 바로 이 회귀분석에서 비롯된 것이 많다는 것을 느꼈다. 
단지 다중회귀분석일지라도 변수선택을 잘하면 굉장히 강력한 모델 생성이 가능한 것 같다.
